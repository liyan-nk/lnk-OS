# LNK OS

Interactive personal portfolio website for Liyan Nechikaden.

Goal:
Create a memorable portfolio experience that feels like a modern Linux terminal.

## Core Stack
- **Build Tool**: Vite
- **Frontend**: React
- **Styling**: Tailwind CSS (custom utility bindings)
- **Animations**: Framer Motion
- **Hosting**: Vercel

## Key Rules
- No backend (fully client-side static)
- No authentication or databases
- High mobile responsiveness with hybrid touch inputs
- Loading speed optimization with low latency and quick boot transitions
- Simplicity and clean documentation first
